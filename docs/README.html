Release Notes - HornetQ - Version 2.4.0 Final


12th December 2013

These are the release notes for HornetQ 2.4.0.Final

For full description of the contents please see the HornetQ project JIRA.

2.4.0.Final is fully JMS 2.0 compliant, other cool new features include single port support for all protocols and mobile push integration via AeroGear.
For more info on what this release has to offer please refer to the quick start guide or the user manual. Enjoy!!
